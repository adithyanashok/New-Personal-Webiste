GymBook - Subscription based gym management app

Project information

  • Name: GymBook
  • Tech Used: React Native, NodeJs, AWS, Docker, Firebase
  • About App:
  • Complete gym management application developed using React Native and NestJS to run gym operations.
  • Features:
  • 1. Member Management: Total member CRM Add, edit, and remove member profiles
  • 2. Expiration Day Alert - Expiration alerts for users.
  • 3. AWS S3 Image Upload - Safe storage of member photos
  • 4. Stripe Integration - Secure payment gateway for app subscriptions.
  • 5. Monthly Statistics - Comprehensive overview by selected month
  • 6. Revenue Trend Analysis - Monthly revenue tracking
  • 7. Twillio OTP Verification - Secure phone verification for owners and members
