Client Project

Wise Banker

A financial assessment platform built to help users understand their financial health and move through a guided service workflow ending in report generation.

Role Full-stack developer
Stack Node.js, Express, PostgreSQL, MongoDB, AWS, Razorpay
Node.js ExpressJS PostgreSQL MongoDB AWS S3 Razorpay PDF Reports

What I built

  • Built the full-stack workflow for user onboarding and financial evaluation.
  • Designed the backend with Node.js and Express using PostgreSQL and MongoDB.
  • Implemented server-side PDF generation for dynamic financial reports.
  • Integrated AWS S3 for secure file storage and Razorpay for payment processing.
  • Delivered an end-to-end journey from user intake to report delivery.

Why it matters

  • Shows client-facing product execution, not just personal projects.
  • Combines data modeling, document generation, and payment handling.
  • Demonstrates backend ownership for a fintech-flavored workflow.
  • Balances technical architecture with a guided user experience.
