Personal Project

GameVera

A scalable game review platform built with microservices, real-time community interaction, AI-generated game insights, and production-focused deployment practices.

Architecture 6 NestJS services with real-time communication
Stack React.js, Tailwind, NestJS, Postgres, MySQL, AWS, Docker
React.js Tailwind NestJS Socket.IO JWT + OAuth Docker CI/CD

What I built

  • Architected six independent services using NestJS.
  • Built a live discussion system using Socket.IO.
  • Integrated Google Gemini to generate insights from user reviews.
  • Implemented JWT and OAuth authentication with CSRF protection and rate limiting.
  • Used caching, Docker, AWS, and CI/CD to improve performance and deployment flow.

Why it matters

  • Shows stronger system design than a standard CRUD portfolio app.
  • Combines frontend product polish with backend service boundaries.
  • Demonstrates security, performance, and deployment awareness.
  • Represents the kind of scalable product work I want to be known for.
